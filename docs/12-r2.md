# 12 — Cloudflare R2

🎯 Object storage เก็บรูป/วิดีโอ/uploads/backup — ไม่กินดิสก์ VPS และ**ไม่มีค่า egress**

🤔 ทำไม R2 ไม่ใช่เก็บบนเครื่อง: ดิสก์ VPS 75GB ตึงเร็วมากเมื่อมี workflow
สร้างรูป/คลิป + ค่า R2 ถูก ($0.015/GB-month, free tier 10GB, egress ฟรี)

## ⚙️ วิธีทำ

1. Cloudflare Dashboard → R2 → สร้าง buckets: `images`, `videos`, `uploads`, `backup`
2. R2 → Manage API Tokens → สร้าง token (Object Read & Write)
3. ใส่ค่าใน `.env`: `R2_ACCOUNT_ID`, `R2_ACCESS_KEY_ID`, `R2_SECRET_ACCESS_KEY`

## ใช้จาก app

R2 คุยด้วย S3 API — Laravel ใช้ disk `s3` โดยตั้ง `AWS_ENDPOINT`
(ดู `templates/laravel/.env.example`), Next.js/n8n ใช้ AWS SDK ชี้ endpoint เดียวกัน

## ใช้เป็น backup target

ติดตั้ง rclone บนเซิร์ฟเวอร์:

```bash
curl https://rclone.org/install.sh | sudo bash
rclone config    # new remote ชื่อ "r2", type s3, provider Cloudflare
```

จากนั้น `backup/scripts/postgres-backup.sh` จะ sync ขึ้น R2 ให้อัตโนมัติ

## 🧪 ทดสอบ

```bash
rclone lsd r2:                      # เห็น buckets
echo test > /tmp/t && rclone copy /tmp/t r2:backup/ && rclone ls r2:backup/
```
